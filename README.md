# LeetCode Copy Helper

一个用于在 LeetCode / 力扣题目页一键复制题目的 Tampermonkey 用户脚本。

## 功能

- 复制题号、标题、链接
- 复制难度、标签
- 复制题目正文、示例、提示
- 输出为 Markdown
- 按钮嵌入题目页工具栏

## 安装

- 安装 Tampermonkey
- 从 Greasy Fork 安装脚本
- 或从 GitHub 下载/安装

## 支持站点

- `https://leetcode.cn`
- `https://leetcode.com`
